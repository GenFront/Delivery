### Функциональность

**Реализовано:**

1. Адаптивная структура макета
Использование Flexbox для построения сетки
Адаптивный контейнер с максимальной шириной 1230px
Минимальная высота страницы равна высоте экрана (100vh)
2. Шапка (Header)
- Логотип компании
- Кнопка авторизации пользователя (с иконкой)
- Кнопка корзины заказов (с иконкой)
- Выравнивание элементов с помощью Flexbox
- Минимальная высота: 124px
3. Основная область (Main)
- Занимает всё доступное пространство между шапкой и футером
- Градиентный фон для визуального разделения
- Готовый контейнер для будущего контента
4. Подвал (Footer)
- Логотип компании
- Навигационное меню с пунктами:
   - Ресторанам
   - Курьерам
   - Пресс-центр
   - Контакты
- Социальные сети (Instagram, Facebook, VK)
- Минимальная высота: 156px
- Гибкое расположение элементов
5. Компоненты интерфейса
- Кнопки с двумя стилями:
- Основная кнопка (primary): синий фон
- Контурная кнопка (outline): белый фон с границей
- Иконки в кнопках для лучшей визуализации

**CSS Методология:**

- Использование CSS Flexbox для всех макетов
- Применение box-sizing: border-box для всех элементов
- Использование CSS-переменных и современных свойств
- Градиенты для фоновых элементов

**Адаптивность:**

- Контейнер адаптируется под разные размеры экрана
- Padding контейнера: 15px с каждой стороны
- Гибкие отступы между элементами
- Поддержка мобильных устройств

Дизайн элементы

**Цветовая палитра:**

Основной синий: rgba(24, 144, 255, 1)
Текст: rgba(89, 89, 89, 1)
Границы: rgba(217, 217, 217, 1)
Фон: rgba(245, 245, 245, 1)
Градиент основного блока

**Типографика:**

Стандартный шрифт системы
Контрастные цвета для читаемости

**Особенности верстки**

1. Flexbox Layout:
- Гибкое распределение пространства
- Выравнивание по центру
- Автоматические отступы

2. Модульность:
- Разделение на независимые блоки
- Переиспользуемые компоненты (кнопки, логотип)
- Изолированные стили для каждого блока
